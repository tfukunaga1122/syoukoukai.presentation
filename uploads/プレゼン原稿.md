## 発表のコツ

### スライドと原稿の使い分け

* **スライドに書いてあることは読まない**
* スライド＝視覚で「見せる」（写真・数字・キーワード）
* 原稿＝聴覚で「語る」（ストーリー・背景・感情）
* 両方で同じ情報を出すと聴衆の注意が分散する

### 宿泊棟・体験施設スライドの心がけ

* 各棟10〜16秒という短さなので、**1スライド1フレーズ**を徹底する
* スライドのアニメーション（フェードイン）が終わるのを待ってから話し始めると、自然なリズムになる
* 読み上げではなく「補足」のつもりで。写真が主役。

### 全体のリズム

```
無言のつかみ → 自己紹介 → 核フレーズ
→ 輝き → 影（↓）→ 再生（↑）
→ 宿泊棟×4（テンポよく）→ 体験施設×4（テンポよく）
→ 数字（落ち着く）→ 核フレーズ回収（静）→ お願い → 余韻
```
